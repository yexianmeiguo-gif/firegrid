# 🛡️ FireGrid 备份与恢复 - 速查卡

## 🚀 一键备份
```bash
fg-backup
```

## 📋 查看备份
```bash
fg-backups
```

## 🔄 恢复（最新）
```bash
fg-restore latest
```
然后输入：YES

## 🔄 恢复（指定版本）
```bash
fg-restore 20260309_214227
```
然后输入：YES

## 🔄 恢复后重启
```bash
openclaw gateway restart
```
