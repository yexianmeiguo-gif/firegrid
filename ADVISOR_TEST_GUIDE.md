# AI 装备顾问测试指南

## 测试环境准备

### 1. 微信开发者工具
- 打开微信开发者工具
- 导入 `firegrid/weixin-miniprogram` 项目
- 确保编译模式为 "正常编译"

### 2. 后端 API 状态检查
```bash

curl https://firegrid-backend-230828-6-1409174142.sh.run.tcloudbase.com/health

# 测试装备搜索接口
curl -X POST https://firegrid-backend-230828-6-1409174142.sh.run.tcloudbase.com/ai/equipment/search \

  -d '{"query":"城市高层消防车","limit":5}'
```

## 问题排查

### 问题 1: 页面显示空白
**检查**：
- app.json 是否正确添加了 `pages/advisor/advisor`
- advisor.json 是否存在且格式正确
- 开发者工具是否清除了缓存

### 问题 2: 查询无响应
**检查**：
- 网络请求是否正常（开发者工具 Network 面板）
- app.js 中的 apiBaseUrl 是否正确
- 后端服务是否在运行

### 问题 3: 模拟数据不显示
**检查**：
- advisor.js 中的 `loadMockData()` 方法是否正常
- 数据格式是否正确
