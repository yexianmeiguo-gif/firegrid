## 🚀 立即开始使用

### 第一步：设置（1分钟）

```bash
# 1. 确保快捷命令已加载
source ~/.openclaw/workspace/firegrid/fg-commands.sh

# 2. 或者添加到启动文件（永久生效）
echo "source ~/.openclaw/workspace/firegrid/fg-commands.sh" >> ~/.zshrc
```

### 第二步：启动Agent团队

```bash
fg-agents start
```

### 第三步：使用

现在你只需要：
1. **在飞书告诉我需求**

## 🎯 3种工作模式

### 模式A：轻度参与（推荐）
- Agent自动工作
- 复杂问题请示你
- 完成时通知你

**适合**: 想省心，但想掌控关键决策

### 模式B：完全放手
- Agent全权处理
- 只汇报关键结果
- 深夜也自动修复

**适合**: 完全不想管技术细节

### 模式C：实时围观
- 每2小时汇报进展
- 你可以随时查看
- 有问题立即通知

**适合**: 想了解过程，学习技术

## 🔄 实际工作流程示例

### 场景：开发优惠券功能

**你说**: "我要做优惠券系统"

**Agent团队**:
```
🎯 规划Agent (10分钟)
   分析需求 → 拆解12个任务 → 生成计划

💻 开发Agent (2小时)
   并行开发:
   - Agent A: 数据库 (30分钟) ✓
   - Agent B: API接口 (1小时) ✓
   - Agent C: 前端页面 (1.5小时) ✓

🐛 修复Agent (5分钟)
   自动修复3个WXML标签错误 ✓

🧪 测试Agent (30分钟)
   运行26个测试用例，全部通过 ✓

🚀 部署Agent (10分钟)
   部署后端 + 更新小程序 + 配置监控 ✓

📱 飞书通知你 (1秒)
   「优惠券系统已完成！访问 pages/coupon/」
```

**总耗时**: 3.5小时（你参与1次）  
**以前耗时**: 2天（你参与15次）

## 🎁 额外福利

启用Agent团队后，你还会得到：

1. **7x24监控**
