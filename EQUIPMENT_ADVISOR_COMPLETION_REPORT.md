```javascript
try {
  const res = await requestPromise()
  // 使用 API 数据
} catch (error) {
  // 自动降级到模拟数据
  this.setData({ equipmentList: this.generateMockEquipments() })
}
```

## 📱 API 集成

### 1. 查询装备
**端点**: `POST /api/ai/query-equipment`

**请求**:
```javascript
{
  query: "采购高层建筑灭火消防车，预算500万",
  limit: 5
}
```

**响应**:
```javascript
{
  recommendation: "中联重科水罐车性价比最高...",
  equipments: [...],
  totalMatches: 25
}
```

### 2. 获取装备详情
**端点**: `GET /api/ai/equipment/:id`

### 3. 发起询价
**端点**: `POST /api/ai/inquiry`

**请求**:
```javascript
{
  equipmentIds: ["eq_1"],
  question: "咨询详细报价...",
  userId: "current-user-id"
}
```

**响应**:
```javascript
{
  inquiryId: "inq_123456",
  suppliers: [...],
  status: "submitted"
}
```

## 🚀 使用流程

### 典型用户场景
1. 用户打开 AI 页面
2. 点击"装备顾问"切换模式
3. 输入需求："采购高层建筑灭火消防车，预算500万"
4. 选择对比数量（保持默认 5 个）
5. 点击"🔍 查询装备"

7. 浏览 5 个装备对比卡片
8. 点击任意装备展开详情

10. 点击"📞 联系供应商"查看联系方式

13. 点击"📋 生成报告"生成采购报告
