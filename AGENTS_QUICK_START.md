# 🤖 FireGrid 多Agent快速入门

## 1分钟启动Agent团队

### 第一步：添加命令
```bash
echo "alias fg-agents='bash ~/.openclaw/workspace/firegrid/fg-agents.sh'" >> ~/.zshrc
source ~/.zshrc
```

### 第二步：启动Agent团队
```bash
fg-agents start
```

你会看到：
```
🤖 FireGrid 多Agent系统启动器
==============================

✅ Agent团队已启动

当前团队配置：
  🎯 规划Agent - 负责需求分析和任务拆解
  💻 开发Agent - 负责编写代码
  🧪 测试Agent - 负责测试验证
  🚀 部署Agent - 负责部署上线

💡 使用方法：
  1. 在飞书告诉我你的需求
  2. Agent团队自动工作
  3. 完成后飞书通知你
```

## Agent团队介绍

| Agent | 职责 | 工作时间 | 特点 |
|-------|------|---------|------|
| 🎯 规划Agent | 分析需求、拆解任务 | 即时响应 | 不出错、不遗漏 |
| 💻 开发Agent | 写代码、修bug | 7x24小时 | 并行开发、速度快 |

## 定制你的Agent团队

### 轻度参与模式（默认）
- Agent自动工作
- 完成时通知你
- 复杂问题请示你

### 完全放手模式
- Agent全权处理
- 只汇报关键结果
- 深夜也自动修复

### 实时围观模式
- 每2小时汇报进展
- 你可以随时查看
- 有问题立即通知
